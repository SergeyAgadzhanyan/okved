# OKVED Game

Небольшой Spring Boot сервис для определения наиболее подходящего кода ОКВЭД по номеру телефона.

Сервис принимает номер телефона, нормализует его и сопоставляет с классификатором ОКВЭД, загруженным из JSON.  
Поиск выполняется по принципу **максимального совпадения по концу строки**

---

## Запуск проекта

```bash
./gradlew bootRun
```

Приложение стартует на порту **8080**.

---

## Запрос к API

```bash
curl --location 'http://localhost:8080/okved?phone=7911128921'
```
